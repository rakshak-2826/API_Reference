Introduction

The Book Generation API offers a robust solution for generating books across diverse topics using advanced language models. This guide covers how to authenticate, access various endpoints, and provides examples to get you started.

API Endpoints

Book Generation

POST /api/generate-book

Required Parameters

Parameter Type Description
api_provider string Specify the API provider. Options: "openai" or "together"
model_version string The specific model version (e.g., "gpt-3.5-turbo", "gpt-4", "llama-2-70b")
subject string The primary subject or theme of the book

Quick Start Guide

Step 1: Get an API Key

Click the button in the Authorization section above to generate an API key.

Step 2: Submit Your Request

Use your preferred programming language or tool to send a POST request to the /api/generate-book endpoint. Include your API key in the header and the necessary parameters in the request body.

Step 3: Process the Response

The API will return a response with a job ID. Use this ID to track the status of your book generation job.

Step 4: Retrieve the Final Book

Once the process is complete, you can retrieve the generated book using the job ID (retrieval endpoint coming soon).
